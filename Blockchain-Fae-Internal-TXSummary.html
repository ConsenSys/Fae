type InputSummary = (ContractID, TXInputSummary) Source #

The InputResults contained the contract ID but the TXInputSummary does not, because we want to print the ID regardless of whether the input summary throws an error, so we include it separately.

type MaterialsSummaries = Vector (String, InputSummary) Source #

Materials, in addition, are given names under the contract that calls them.
