Description

The primary motivating example of a smart contract is a currency such as Bitcoin. This module provides a typeclass for currency values that can be safely exchanged through the Fae smart contract system.

Basic numeric currency

This is a nearly featureless currency that most likely suffers from the effects of inflation, since its reward function creates one coin at a time. Not suitable for complex economic situations.

type Coin = EscrowID Token CoinVal Source #

This is the actual currency; no user ever looks inside directly.

reward :: MonadTX m => RewardEscrowID -> m Coin Source #

A contract to claim system rewards in exchange for Coin values. The one-to-one exchange rate is of course an example and probably not actually desirable.

Note that there is no Currency instance for rewards: thus, rewards are a unary counting value and can only be collected individually; in addition, claimReward is the only means of spending them, and it destroys the reward token. So this function can be seen as reifying rewards as a true currency, albeit one that cannot necessarily be used to claim rewards from anywhere else.
