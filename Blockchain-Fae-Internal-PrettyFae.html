Blockchain.Fae.Internal.PrettyFae

Description

A class for pretty-printing Fae storage. This has to be done in a FaeStorageT monad, so the class can't just be Pretty.

    showTransaction :: (MonadCatch m, MonadIO m) => TransactionID -> FaeStorageT m c String Source #

    Convenience function for neatly showing a TransactionEntry by ID, rather than actually going into the storage to get and format it. It catches all exceptions thrown by contracts or transactions and prints an error message instead; thus, whatever actually did complete is part of the output.
