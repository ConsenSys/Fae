Description

This module both nicely bundles Control.Monad.Coroutine and Control.Monad.Coroutine.SuspensionFunctors, but it also defines some much-needed mtl instances for Coroutines. Sadly, since monad-coroutine dropped the ball here, these have to be orphan instances (and undecidable to boot), because wrapping Coroutine in a newtype turns out to be surprisingly messy due to its recursive definition.
