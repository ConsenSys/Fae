Description

This module extends the types of Crypto.Hash and Crypto.PubKey.Ed25519 from cryptonite to be easily usable for Fae, and defines some convenience functions for them.

      (GSerializePut (ERep1 f), GSerializeGet (ERep1 f)) => Serialize (SERep1 f) #

      The possibility of writing this instance is why we use the intricate ERep-based substitution scheme rather than recursively descending into the plain Rep, serializing along the way and substituting when necessary without creating a new type. The generic Serialize functions are either trivial (so not worth duplicating) or clever (so hard to duplicate without extensive copy-pasting, and not desirable to replace with something less clever), so it is worth our while to be able to use them.
