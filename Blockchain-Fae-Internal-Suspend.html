Actions

suspend :: Monad m => b -> SuspendT a b m a Source #

This allows you to mark the boundaries of the successive runs of the function. The current call will end when it reaches a suspend, returning the argument to suspend, and evaluating to the argument of the next call.

terminate :: Monad m => b -> SuspendT a b m b Source #

This allows you to mark the end of the computation, returning a value immediately and ignoring subsequent code.
